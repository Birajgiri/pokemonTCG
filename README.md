# Pokemon Trading Card Game Web App

A Flutter web application for browsing and viewing Pokemon trading cards using the Pokemon TCG API.

## Features

- Welcome screen with custom background image and logo
- Grid layout displaying Pokemon cards (4 cards per row on desktop, 2 on mobile)
- Alphabetically sorted card collection
- Click-to-expand card view with detailed information
- Responsive design for desktop and mobile devices
- Card details include HP, Type, Subtype, Rarity, Artist, Set, Series, and Number
- Local storage caching for faster loading
- Smooth Hero animations between views
- Back button to return to grid view

## Technologies Used

- Flutter 3.35.5
- Dart 3.9.2
- Pokemon TCG API v2
- Provider for state management
- SharedPreferences for local storage
- CachedNetworkImage for image optimization

## Development

This project was developed using other trading card game applications as references for UI/UX design and optimized with AI assistance for code quality and performance.

## Running the App

```bash
flutter run -d chrome
```
