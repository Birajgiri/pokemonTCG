# Pokemon Trading Card Game Web App

A Flutter web application for browsing Pokemon trading cards and battling with random Pokemon using the Pokemon TCG API.

## Features

- **Welcome Screen** - Entry page with logo and two navigation buttons (Trading Cards / Pokemon Battle)
- **Trading Cards** - Grid layout with 100 cards, click to expand for details, alphabetically sorted
- **Pokemon Battle** - Random HP-based battles with visual winner indicators on gradient background
- **Responsive Design** - Optimized for desktop, tablet, and mobile devices
- **Local Caching** - Faster loading with SharedPreferences

## Tech Stack

- Flutter 3.35.5 / Dart 3.9.2
- Pokemon TCG API v2
- Provider (state management)
- CachedNetworkImage

## Development

Developed using TCG apps as UI/UX references and optimized with AI assistance.
